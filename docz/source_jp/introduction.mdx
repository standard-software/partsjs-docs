# Introduction

## Parts.js 

- Parts.js はJavaScriptの型安全で多くのプラットホームに対応したライブラリです。

- 誰もが「こんなのあったら便利なのにな」と考えるような関数を多数実装しています。

- コードは可読性を高くしているので、実装の参考にしたり実装をコピペして利用することも容易です。
- Parts.js は他のライブラリに依存せずシンプルに実装されています。


- コードは契約プログラミング手法、あるいは、防御的プログラミング手法を用いているので、型をミスした引数呼び出しを行う場合にはエラーを出力します。
